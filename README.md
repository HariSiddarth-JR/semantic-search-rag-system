# Semantic Search and RAG System

## Overview
This project implements a Semantic Search and Retrieval-Augmented Generation (RAG) pipeline for product review analysis.

Traditional keyword-based search systems fail when users express intent using different words than those stored in the database. This project addresses that challenge using sentence embeddings, vector similarity search, and RAG.

## Features

- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Sentence Embeddings using Sentence Transformers
- FAISS Vector Indexing
- Semantic Search
- Retrieval-Augmented Generation (RAG)
- Precision@K and Recall@K Evaluation
- Business Insights Generation

Dataset

## Results

The system successfully retrieves semantically relevant reviews based on user intent and generates context-aware answers using a Retrieval-Augmented Generation pipeline.
